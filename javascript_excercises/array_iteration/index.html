Array Iteration

Looping through the array and doing something to each element

  • Use a for loop to iterate over an array
    • To loop over an array using a for loop, we need to make use of the arrays length property.
    • var colors = ["blue", "green", "orange", "yellow"];
      for(var i = 0; i < colors.length; i++;){console.log(colors[i]);}
  • Use forEach to iterate over and array
    • built in way to iterate over an array
    • colors.forEach(some function);
    • color argument in the anonymous function is just a placeholder call it whatever you want. function applied to each element in the array
    • colors.forEach(function(color){console.log(color)});
  • Could use a while loop to iterate but more syntax
    • var count = 0; while(count < colors.length){console.log(colors[count]); count++;}
