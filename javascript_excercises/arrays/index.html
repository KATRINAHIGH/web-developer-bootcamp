Arrays

The JavaScript Array object is a global object that is used in the construction of arrays; which are high-level, list-like objects.

Create an array
var myArray = ["basket", "corn", "market"];
var emptyArray = [];

Access an array
first index: myArray[0]; //basket last index: myArray[myArray.length-1]; //market

Arrays can hold any type of data.
var dataArray = [49, true, "friend", null];

Arrays have a length property
var anotherArray = [6, 7, 8, true]; anotherArray.length //3

Array Methods

  • .push() pushes to end of the array
  • .pop() removes from the end of the array
  • .shift() pushes to front of the array
  • .unshift() removes from the front of the array
  • .indexOf() returns the index of an item in an array, returns -1 if not present
  • .slice() use to copy parts of an array, must store in a variable, returns an array of the sliced elements, its not inclusive returns first index and 1 less of the index passed in.
    var num = [5, 6,7,8,9,10,11,23,54] var newArray = num.slice(2,5);
