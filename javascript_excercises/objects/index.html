Objects

  1. Stored in key value pairs. var person = {name: "Travis", age: 21, city: "LA"}
  2. Unlike arrays, objects have no order
  3. To retrieve data out of an object. Use object name followed either by brackets or by dot notation and the key for the value you want to retrieve. person["name"]; or person.name;
  4. There are 3 differences between the types of notation
    • You can not use dot notation if the property starts with a number. ex: someObject.1blah; //invalid someObject["1blah"]; //valid
    • You can look up using a variable with bracket notation. ex. var str = name; someObject.str; //doesn't look for name someObject[str]; evaluates str and looks for name
    • Can't use property names with spaces with dot notation> someObject.fav color //invalid someObject["fav color"]; //valid
  5. Updating Data in an object. Just like an array we can access a property and reassign it.
    var person= {name: "Travis", age: 21, city: "LA"};
    to update age: person["age"] += 1;
    to update city: person.city += "London";
  6. Creating Objects Like arrays there are a few methods of initializing objects
    • Make an empty object then add to it
      var person = {}; person.name = "Travis"; person.age = 21; person.city = "LA";
    • all at once Object Literal Notation
      var person = { person: "Travis", age: 21, city: "LA"};
    • another way of initializing an object
      var person = new Object(); person.name = "Travis"; person.age = 21; person.city = "LA";
  7. Objects can hold all sorts of data..numbers, strings, arrays, boolean, even another object
  8. Nested Arrays and Objects, can nest objects in arrays and arrays in Objects
